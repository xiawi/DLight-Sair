There are currently 9 characters in the game that use bow.

Attacks

NLight

The bow user fires a diagonal arrow shot. Hits grounded as well as airborne targets.

SLight

The bow user swings the bow forward, catches a target and sends them forward with an arrow.

DLight

The bow user shoots the ground in front of them.

Nair

The bow user flips in the air. Catching a target sends them upward with an arrow.

Sair

The bow user swings their bow forward.

Dair

The bow user shoots an arrow diagonally downwards.

Recovery

The bow user swings their bow diagonally upwards. Catching an opponent causes the bow user to shoot them diagonally upwards.

Ground Pound

The bow user aims their bow downwards. Hitting a target causes the bow user to shoot three arrows downwards.
